I'm Delaying the Sealubbers Demo Release

Posted December 4, 2025 by Sealubbers

I originally aimed to release the demo by the end of 2025. Over the last stretch of development, I've run into more delays than expected and I haven't hit the deadlines I set for myself.

I don't want to give a specific replacement date until I'm sure I can actually deliver it. Right now the priority is finishing core systems properly, improving performance, and making sure the demo feels stable and polished when it arrives.

Thank you for sticking with the project and for all the support so far. I'll share the next milestone as soon as I can commit to it with confidence.
